# Cork Board To-Do

A drag-and-drop sticky note to-do board with category-sorted supply, a fixed cork board, and a trash can for completed tasks.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually http://localhost:5173).

## Features

- Drag sticky notes from the supply panel onto the 846×600 cork board
- Reposition notes freely on the board
- Drop notes on the trash can when done
- Preset categories (Work, Personal, Shopping, Ideas) plus custom categories
- Full-screen coastline background; board sits above the fold
- Persists notes and categories in localStorage
